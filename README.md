# 📦 PikPak Enhancement Master

> 🚀 A cloud drive manager for PikPak web, giving the web version operations comparable to a local file system.

## ✨ Key Features

### ✨ Experience & Navigation Engine

* **Interface Redesign**: Built like a Windows File Explorer, greatly enhancing user experience.

* **Speed Mode**: Replaces web logic and blocks native sync (recommended in settings).

* **Advanced Path Bar**: Supports scrolling, dropdown switching, path echo, and breadcrumb navigation.

* **Experience Enhancements**:

  * Dark mode
  * Video duration display
  * One-click blur cover (privacy protection)
  * Support for starred files & file type sorting

* **Background Auto-Indexing**: Blue breathing dot on homepage icon indicates directory sync status.

---

### 📂 Batch & Storage Management

* **Batch Rename**

  * Regex replace/delete
  * Series numbering
  * Text formatting (case / full-width / half-width)
  * Standard naming (e.g., numbering / FC2)
  * Prefix ad removal
  * MIME extension repair

* **Analysis Suite**

  * File analysis (type filter + duplicate check: hash / duration / name)
  * Folder analysis (size filter + duplicate check: name / similarity / coverage)
  * Supports exporting directory tree

* **Smart Organization**

  * Remove empty folders
  * Batch unzip (auto-fill password + auto-delete after extraction)

* **Resource Manager**

  * Blacklist: quickly remove junk files
  * Whitelist: protect files from bulk deletion
  * Custom rules available in settings

> ⚠️ Avoid modifying data from multiple devices simultaneously.

---

### 🌐 Transfer & Sharing Center

* **Share Management**: Supports limited extraction counts and automatic unsharing.

* **Offline Download**: Batch offline tasks + link export.

* **Fast Upload**

  * Supports large files
  * Greatly reduces small file upload failures

> ⚠️ Extraction limits rely on front-end logic and only work if the web page is active and the device is not asleep.

---

### 🎬 Immersive Media Enhancements

* **Playback Engine**

  * 0.5x – 3.0x speed
  * Rotate / mirror / adjust aspect ratio
  * Auto skip intro/outro
  * Continuous / loop playback
  * Timeline thumbnail preview

* **Subtitle System**

  * Cloud / local / online search
  * Personalized subtitle settings
  * Drag & drop subtitle loading

* **Visual Assistance**

  * Image search (images / video frames)
  * Media mode (automatic A-Z sorting for series / comics)

---

### ⚙️ Configuration & Data Management

* **Configuration Backup**

  * JSON export
  * Cross-device migration supported

* **Data Cleanup**

  * Full index / preferences / management rules / password vault / history

> 📌 Full index is temporary; other data is persistent.

---

### ⚡ Download & Distribution

* **External Direct Link**

  * Supports RPC push to Aria2 / Motrix

* **Download Filter**

  * Filter files by extension or keywords

## 📥 Installation Guide

### 1️⃣ Install a Userscript Manager

* 👉 [Tampermonkey](https://greasyfork.org/ko/scripts/556685)

### 2️⃣ Install the Script

* Click to install [`PikPak_Enhancement_Master.user.js`](https://github.com/digbug82/PikPak_Enhancement_Master/blob/cffc9e5d9a72c3ae1063305572b62be9a607a03d/PikPak_Enhancement_Master.user.js)  
* Or manually import the script into the extension

### 3️⃣ Open [PikPak Web](https://mypikpak.com/drive)

### 4️⃣ Start the Script

After refreshing the page, click the **sidebar floating ball** to activate ✅
